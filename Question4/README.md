1.
Command:
uptime

Explanation:
This command shows how long the system has been running since last boot.

2.
Command:
ps -u $USER

Explanation:
This command lists all processes running under my user account.

3.
Command:
top

Explanation:
This command helps to identify which process is using highest cpu.

4.
Command:
sleep 300 &

Explanation:
This command runs a process in background.

5.
Command:
jobs

Explanation:
This command verifies that the background process is running.

6.
Command:
renice 5 -p some pid value

Explanation:
This command changes process priority. Negative values are not allowed for normal users.

7.
Command:
free -h

Explanation:
This command displays memory usage in human readable form.

8.
Command:
df -h ~

Explanation:
This command shows disk usage of the filesystem where home directory exists.

9.
Command:
echo $SHELL

Explanation:
This command displays the name of shell currently in use.

10.
Command:
uname -a > system_report.txt

Explanation:
This command redirects system information output into a file.

11.
Command:
ncdu ~

Explanation:
This command shows disk usage in interactive way and helps find large folders.
