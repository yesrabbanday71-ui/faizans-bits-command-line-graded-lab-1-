1.
Command:
whoami

Explanation:
This command shows my current login username. It confirms that i am working with my own user account.

2.
Command:
groups

Explanation:
This command displays all the groups my user belongs to in the system.

3.
Command:
pwd

Explanation:
This command prints the present working directory where i am currently located.

4.
Command:
ls -l

Explanation:
This command lists all files and folders with detailed info like permissions and ownership.

5.
Command:
echo "Linux user environment verified" > user_info.txt

Explanation:
This command creates a file and writes a confirmation message into it.

6.
Command:
wc -c user_info.txt

Explanation:
This command counts the total number of characters present in the file.

7.
Command:
man mkdir

Explanation:
This command opens the manual page for mkdir. The -p option is useful because it creates parent directories if needed.

8.
Command:
ls ~ | sort

Explanation:
This command lists home directory contents and sorts them in alphabetical order.

9.
Command:
grep "admin" log.txt

Explanation:
This command searches for the word admin inside log.txt and shows matching lines only.

10.
Command:
uname -r

Explanation:
This command displays the current running Linux kernel version.

11.
Command:
ping -c 4 www.google.com

Explanation:
This command checks network connectivity by sending ICMP packets to google.

12.
Command:
uptime

Explanation:
This command shows how long the system is running, number of users and load average.
